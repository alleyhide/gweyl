## Requirement

The followings are required.
boost 1.62 (or more)
cmake

For example, if you use Debian, do
$ sudo apt install libboost-all-dev cmake

## Usage

    #include "gweyl.hpp"

## COMPILE

Do
$ cmake .
$ make

If you want to test, do
$make test.
